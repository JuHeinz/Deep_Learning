Language Model zur Wortvorhersage mit rekurrenten LSTM Network

Wortvorhersage Diskussion Dokumentation

Wortvorhersage

Diskussion

Als Anzahl der Trainings-Epochen wurden 20 gewählt, da der loss ab ca. der 20. Epoche abflacht.

Es ist zwar möglich den Original-Text zu rekonstruieren, allerdings ist das original nachfolgende Wort meistens nicht der Voschlag mit der größten Wahrscheinlichkeit. Dies liegt wohl vor allem daran, dass die von mir gewählte Sequence Length sehr gering (5) ist. Das heißt, die Vorhersage erfolg nur an Hand der letzen 5 Wörter. Datenschutz Probleme könne natürlich tortzdem entstehen, so zum Beispiel bei Namen, da der Nachname einer Person in den meisten Fällen direkt auf deren Vornamen folgt.

Dokumentation

Modell-Parameter:

  • Stacked LSTM: 2 hidden Layer (in sich rekursiv) mit je 100 LSTM Units
  • Softmax Output mit der Dimension des Dictionaries
  • Loss Function: Cross-Entropy
  • Optimizer Adam mit Learning Rate = 0.01 und Batch Size = 32

Das Modell wurde auf dem ersten Kapitel des Romans Emma von Jane Austen trainiert. (Vobabulary Size = 839)

Folgende Libraries werden auf dieser Seite benutzt:

  • TensorFlow for JavaScript für das rekurrente LSTM-Network als Modell zu definieren, dieses zu trainieren und Predictions mit diesem durchzuführen.
  • TensorFlow Visor für die Darstellung des Trainingsfortschritts.
  • Bootstrap 5.3.5. als CSS-Framework.
